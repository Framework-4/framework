How to use the grid

Every page should have a div with a class of "container" that is used to center the page in the browser window. The header, the optional banner, the "gridContainer" and the footer are all inside this div.

The header contains the nav and the logo.

Next comes the "gridContainer" div. Everything except the footer should be contained in this div. Inside this div, you can have semantic sections like "main" and "aside", etc.

But inside these sections you must have a structures of divs with a class or "row" enclosing divs with a column classes like this: class ="col-s-6 col-m-4 col-l-3". This means the div would take up 6 columns at small size, four columns at medium size, and 3 columns at large size.At very small size (mobile portrait), all divs default to 100%.

If you want a section to be full width at all breakpoints, give it a SINGLE class of "col-s-12".

Each row at each width should add up to 12 columns or multiples of 12 columns.
