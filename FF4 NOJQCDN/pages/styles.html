How to use our styles

Framework-4 has taken the pain out of applying css styles. With our handy classes, you can format your pages quickly. No long will you have to write CSS code like this: 


    header h2{
      font-style: italic;
      font-weight: bold;
      text-transform: uppercase;
    }

Instead, you can quickly apply a class of "italic", "bold", or "uppercase" to your element and our Framework-4 stylesheet will take care of the rest. Click on the buttons below to for more about how to use our framework styles.

Our stylesheet also makes typography simple by using rems for font sizes. The html is set to 62.5% so that all the REM measurements are based on 10px sizing. So basically 1.5rem = 15px. Here are our basic settings and they can be easily adjusted for the font families you choose. 


html {
	font-size: 62.5%;
}
body {
	font-size: 1.5rem; 
	line-height: 1.5;
	font-weight: 300;

h1 { font-size: 3.6rem; line-height: 1.25;}
h2 { font-size: 3.0rem; line-height: 1.3; }
h3 { font-size: 2.4rem; line-height: 1.35;}
h4 { font-size: 1.8rem; line-height: 1.5; }
h5 { font-size: 1.5rem; line-height: 1.6; }
h6 { font-size: 1.2rem; line-height: 1.0; }

Framework-4 has styles that can be applied to four kinds of buttons: the HTML button tag, the Submit button, the Reset button, and the input with a type of button. There are styles for four sizes of buttons as well as square, rounded-corner, and oval button shapes.

Tables

Tables have a width of 100%. If you want to restrain the size, adjust the number of columns they span or give the a class of "halfWidth." Tables should be constructed with standard HTML tags like this: 


<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th> 
    <th>Location</th>
    <th>Salary</th>
  </tr>
   <tr>
    <td>Sam</td>
    <td>Gomez</td>
    <td>Portland</td>
    <td>$76,000</td>
  </tr> 
   <tr>
    <td>Alice</td>
    <td>Hammer</td>
    <td>Portland</td>
    <td>$38,000</td>
  </tr> <tr>
    <td>Ruth</td>
    <td>Ginsberg</td>
    <td>Portland</td>
    <td>$46,000</td>
  </tr>
</table>

Do you want to float an element left or right? Just give it a class of "floatLeft" or "floatRight" and add a class of "clearfix" to the containing element. Do you want your images to be responsive? Give them a class of "img-responsive".

Do you want to set sections of your html page to be non-printing? Use the class of "noPrint" on those sections. We have many other useful classes. Please browse our CSS to find classes that will shorten your coding time and increase your coding pleasure.
