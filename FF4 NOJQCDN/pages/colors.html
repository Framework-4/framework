How to use our colors

Designers who use SASS often include variables for colors and fonts. The team at Framework-4 wanted to give its users the same convenient tools without the need compile the code into CSS. YOu can copy and and paste the CSS code below to define your variables, edit the colors, and then use them throughout your site as shown in the example. In naming the variables, we found that names like "light," "medium," and "dark" worked for some palettes, but not for others and the same was true with names like "primary-color" and "secondary-color." So to keep things simple, we named our variables "--color1", "--color2", etc. For our Framework-4 site we used our "Seaglass" palette. You can see this color palette and others by clicking the button below. 


 :root {
  --color1: #000;
  --color2: #303030;
  --color3: #606060; 
  --color4: #D3D3D3;
  --color5: #fff;
	

 body{
  background-color: var(--color2);}
header, footer{
  background-color: var(--color4);}
.container{
  background-color: var(--color5);}

This is the "Seaglass" palette that we used in contructing this site. We liked the range of cool greens and the conrasting deep blue (almost black). Although our Framework-4.css ships without variables and uses a neutral palette of black, white, and shades of gray, it is easy to define variables at the top of the file and choose from some of our marvelous color palettes.
