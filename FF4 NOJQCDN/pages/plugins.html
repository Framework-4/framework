How to use our plugins

When our team was creating Framework-4, we made a commitment to offer to our users any components that we used on the site, plus any additional ones that we thought might be useful. For example, our site uses a dropdown navigation at small screens that converts to a horizontal one at large screens, and so we offer our users a navigation plugin.

Also, on some pages that would otherwise run long, we use collapsible panels to show and hide information. And so we also offer a collapsible-panel plugin. And although we don't use a slide show on this site, we thought our users might like to have one, so we've included one. Click on the buttons below to preview our great plugins.

Our navigation plugin comes as a set of three components: a basic HTML page with the navigation elements in place, a CSS page needed to convert the links from vertical to horizontal and make them look beautiful, and a JS file needed to drop the navigation down on small screens. The plug-in is a part of our standard download but you can see a preview here:

The Framework-4 site is designed to present a lot of information and make it easily accessible, so we did not have a need for an image slider or carousel. But we wanted to make one available to our users. The slider is a part of our standard download and you can see a preview here:

As you can see, one plugin that we found very useful on this site was the collapsible panel plugin. By using it, we were able hide information and make it available only when you wanted to see it and clicked on the button. This plugin uses pure CSS and we know you will find a use for it.You can see a preview of the page you will get in our standard download here:

Although they are a necessary part of any web designer's life, writing and styling HTML forms is not always the most fun part. We wanted to help, so we offer a fully-formatted single-page form that can be adapted to almost any use. And you can also copy and and paste sections into another page and use the styles that come with our framework or overwrite them with your own. For example, if you need a form with checkboxes, you can copy and paste this code and edit the content to fit your needs. 


<label for='feature'>
  <span>Checkbox:</span>
  <ul class ="checkboxList">
  <li><input type="checkbox" name="feature" value="feature1"> Feature 1 </li>
  <li><input type="checkbox" name="feature" value="feature2" checked> Feature 2 </li>
  <li><input type="checkbox" name="feature" value="feature3"> Feature 3 </li>
  <li><input type="checkbox" name="feature" value="feature4"> Feature 4 </li>
  <li><input type="checkbox" name="feature" value="feature5"> Feature 5 </li>
  <li><input type="checkbox" name="feature" value="feature6"> Feature 6 </li>
  </ul>
</label>
