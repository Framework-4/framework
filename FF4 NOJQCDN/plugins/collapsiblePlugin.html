How to create collapsible containers

To create collapsible containers, you need to have button with a class of "collapsible" and a div underneath with a class of "content." Make sure you include the style in the head above or copy and paste it to a linked css file. For this demo, the container has been given a max-width 700px and the main section has been given left and right margins, just to make it easer to see the layout.

Inside any semantic sections in the "gridContainer" you must have divs that have row and column classes.You can change the number of columns a div will occupy at different browser widths ("breakpoints"). These are set by media queries in the CSS and can be changed to suit your layout. It is good practice to make sure that the number of columns occupied on a row add up to 12 or multiples of 12 at every breakpoint.
