Navigation plugin

To use this navigation plugin, simply save as your index page and edit the links. THe javasccript is included at the bottom. You will need to link to the navPlugin.css file as shown in the head.You will also need to link to a JQuery library as shown in the head. Make sure you correct the paths for the css files and the js file. If you want to use a banner image,you can add it underneath the header.

A few embedded styles, like body color and container size have been added to the head section just to make this demo easier to see.
